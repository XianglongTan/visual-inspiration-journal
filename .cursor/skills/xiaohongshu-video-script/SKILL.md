# 小红书爆款短视频脚本生成器

## 核心算法与原则

1. **30秒法则**：必须在黄金前3秒留住用户，并确保内容能让用户至少停留30秒，否则算法会降权。
2. **搜索SEO导向**：小红书是"生活方式搜索引擎"，脚本中需自然融入5-8个高频搜索关键词和相关话题标签。
3. **双高价值驱动（核心）**：小红书极度看重**收藏率**与**点赞评论率**。内容必须具备以下两种价值之一（或结合）：
   - **实用价值（干货）**："利他性"、强"信息密度"，让用户觉得"有用"。
   - **情绪价值（共鸣）**：提供"向往感"（审美/生活方式）、"陪伴感"（嘴替/吐槽/真实经历）或"治愈感"，让用户觉得"懂我"或"想成为"。

## 脚本黄金三段式公式

所有小红书爆款脚本必须遵循 **钩子(Hook) + 价值释放(Value/Emotion) + 闭环(CTA)** 的结构：

### 1. 钩子段（前0-5秒）
**目的**：打破预期、制造冲突、引发好奇、精准圈层。
- **情绪+冲突公式**：目标人群痛点 + 反差认知 + 解决方案。
- *常用句式*：
  - "全网都在夸的XXX，我居然踩雷了？"
  - "你以为...其实大错特错！"
  - "听我一句劝，不要再瞎做XXX了，教你一招..."
  - "月薪3千和月薪3万的XXX，到底差在哪？"

### 2. 价值释放段（5-45秒）
**目的**：提供高密度、结构化的干货，或者输出极强的情绪共鸣点。

**路线 A：实用价值类（干货/教程）**
- 语速要快（建议250-300字/分钟），不要废话。
- **倒金字塔模型**：结论先行 -> 拆解步骤1、2、3 -> 补充底层逻辑或避坑指南。
- **FAB产品模型**：特点(Feature) -> 优势(Advantage) -> 利益(Benefit)。

**路线 B：情绪价值类（生活方式/观点/吐槽/治愈）**
- **嘴替共鸣模型**：场景重现（遇到奇葩同事/极品甲方） -> 犀利吐槽/反击观点 -> 价值观升华（"不要内耗，要发疯"）。
- **向往感模型**：沉浸式视听体验（如ASMR/唯美空镜）+ 娓娓道来的生活感悟文案，营造"理想生活切片"。
- **逆袭/成长故事模型**：低谷起点 -> 艰难摸索的一个转折点 -> 现在的状态对比。重点在于**真实坦诚**，展现"我也曾和你一样"。

### 3. 闭环段（最后5-10秒）
**目的**：引导用户进行点赞、收藏、评论，拉升互动数据。
- **引导收藏**："怕用的时候找不到，建议先点赞收藏"、"完整思维导图放在视频最后了，记得截图保存"。
- **引导评论**："你的情况是哪一种？在评论区告诉我，我帮你参谋"、"大家觉得这套方法有用吗？"
